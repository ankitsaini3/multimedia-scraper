## Purpose

Multimedia Scraper is a production-grade modular multimedia acquisition and playback platform.

The system supports:

- multimedia search
- metadata extraction
- stream extraction
- playback
- downloading
- plugin-based providers
- streaming orchestration
- future browser automation
- future indexing and persistence
- future distributed execution

The project is intentionally designed for long-term growth.

The architecture prioritizes:

- stability
- explicit contracts
- plugin isolation
- incremental evolution
- integration safety
- scalability
- maintainability

---

## Core Principles

1. Architecture first.
2. Stable contracts over rapid rewrites.
3. Extension over replacement.
4. Incremental evolution.
5. Strict layering.
6. Explicit integration boundaries.
7. Test shared systems aggressively.
8. Protect plugin APIs.
9. Avoid hidden coupling.
10. Preserve backward compatibility.
